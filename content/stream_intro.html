Introduction to this blog

This is an example of how a blog can be implemented in this web application framework. It basically consists of three files:

#File nameDescription
1blog_intro.htmlThis HTML code
2blog_footer.htmlThe code at the end of any blog page
3blog_pages.csvA list of blog entries

The list file has each entry on a separate line, with the following values separated with commas:

#Entry nameDescription
1fileThe entry file name
2titleThe title of the entry
3dateWhen the entry was created
